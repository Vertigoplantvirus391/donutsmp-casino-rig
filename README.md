### 2) Download the release

### 3) Open your Minecraft mods folder

If you already have Fabric set up:

- Press `Windows + R`
- Type `%appdata%\.minecraft`
- Press Enter
- Open the `mods` folder

If you do not see a `mods` folder:

- Open `.minecraft`
- Right-click
- Create a new folder named `mods`

### 4) Move the file

- Copy the downloaded mod file into the `mods` folder
- Do not unzip it unless the release page says to
- Keep only the release file in the folder unless you know you need more mods

### 5) Launch Minecraft

- Open your Minecraft launcher
- Pick your Fabric profile
- Make sure the game version matches the mod version
